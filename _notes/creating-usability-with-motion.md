Realtime interactions
can also be though of as **‘direct manipulation,’** in that the user is interacting with the interface objects directly and immediately. The behavior of the interface is happening as the user is using it.

Non-realtime interactions 
**Happen only after input from the user** and have the effect of briefly locking the user out of the user experience until the transition completes.
